# ระบบบันทึกผลและลงทะเบียนรายบุคคล

## Laudato Si' 3D Mission

เอกสารนี้ใช้ประกอบการเชื่อมเกม `index.html` กับ Google Sheet ผ่าน Google Apps Script เพื่อเก็บข้อมูลนักเรียน คะแนน เหรียญ คำตอบ และรายงานสรุปรายบุคคล

## ข้อมูลที่นักเรียนกรอกในเกม

| รายการ | รายละเอียด |
| --- | --- |
| ชื่อนักเรียน | ชื่อผู้เล่นรายบุคคล |
| ห้อง | เช่น ม.6/1 |
| เลขที่ | เลขที่ในชั้นเรียน |
| ฉายาผู้พิทักษ์ | ชื่อเล่นหรือฉายาที่ใช้ในเกม |

## ข้อมูลที่ระบบบันทึกลง Google Sheet

| หมวด | ข้อมูล |
| --- | --- |
| ข้อมูลผู้เล่น | ชื่อ ห้อง เลขที่ ฉายา |
| คะแนน | คะแนนรวม คะแนนเต็ม ระดับ คะแนนแต่ละภารกิจ |
| เหรียญ | เหรียญที่ปลดล็อกได้จากแต่ละภารกิจ |
| คะแนนครู | ความรับผิดชอบและความคิดสร้างสรรค์พิเศษ |
| หลักฐานการเรียนรู้ | คำตอบทั้งหมดในรูปแบบ JSON และรายงานข้อความ |
| คำมั่นสัญญา | ภารกิจจริงที่นักเรียนเลือกปฏิบัติหลังจบเกม |

## วิธีตั้งค่า Apps Script

1. เปิดโปรเจกต์ Apps Script ที่ต้องการใช้
2. วางไฟล์ `Code.gs` จากโฟลเดอร์นี้ลงในโปรเจกต์
3. สร้างไฟล์ HTML ชื่อ `Index` แล้ววางโค้ดจาก `index.html`
4. กด Run ฟังก์ชัน `setupSheet` เพื่อสร้างชีต `LaudatoSi3DMission`
5. Deploy เป็น Web App
6. ให้นักเรียนเปิด Web App URL เพื่อเล่นเกมและส่งผลคะแนน

## หัวตารางที่สร้างโดย `setupSheet`

`Timestamp`, `ชื่อนักเรียน`, `ห้อง`, `เลขที่`, `ฉายาผู้พิทักษ์`, `คะแนนรวม`, `คะแนนเต็ม`, `ระดับ`, `ภารกิจที่สำเร็จ`, `เหรียญ`, `ความรับผิดชอบ`, `ความคิดสร้างสรรค์พิเศษ`, `Wake Up Call`, `Eco Detective`, `Mission Possible`, `Create & Launch`, `Pitch for the Planet`, `Real Life Quest`, `คำมั่นสัญญา`, `คำตอบทั้งหมด JSON`, `รายงานข้อความ`

## หมายเหตุการใช้งาน

ถ้าเปิด `index.html` จากไฟล์ในเครื่อง ระบบจะบันทึกข้อมูลไว้ในเครื่องก่อน ครูยังสามารถดาวน์โหลดรายงานเป็นไฟล์ข้อความได้ ส่วนการส่งเข้า Google Sheet จะทำงานเต็มรูปแบบเมื่อเปิดผ่าน Apps Script Web App หรือเมื่อใส่ Web App URL ลงในตัวแปร `APPS_SCRIPT_WEB_APP_URL` ใน `index.html`
